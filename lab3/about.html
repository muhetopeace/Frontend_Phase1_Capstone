About Book Explorer

Book Explorer is a responsive web app that allows users to search, browse, and save their favorite books using the Open Library API.

Mission: Our mission is to make book discovery fun, easy, and accessible for everyone.

Technologies: HTML, TailwindCSS, JavaScript (Async/Await & fetch API), and localStorage.

How to Use: Go to Home, search or browse books, and click "Add to Favorites" to save books you like. Visit Favorites page to see all saved books.

This app was created as a learning project to practice asynchronous JavaScript, API integration, DOM interactivity, and persistent data storage.
